# MediTrack • Medication Management App

A modern, responsive single-page web prototype built for medication tracking, smart reminders, healthcare automation, personalized diet plans, and patient education.

Designed with Indian users in mind — supports **English, हिंदी, and తెలుగు**.

![MediTrack Hero](https://via.placeholder.com/800x400/0f1b2d/ffffff?text=MediTrack+Hero)  


## ✨ Key Features

- Add & manage medications with expiry tracking
- Smart reminders system
- **Automation Studio** — build custom workflows (refill alerts, caregiver escalation, hydration nudges, etc.)
- Medicine information checker
- Personalized diet planner (Veg / Non-veg / Vegan + health conditions)
- Health education articles + adherence progress dashboard
- Emergency quick access (108 + common OTC medicines)
- Beautiful teal-navy UI with smooth animations
